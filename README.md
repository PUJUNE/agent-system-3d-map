# 에이전트 시스템 3D 지도

로컬 폴더 상주형 AI 에이전트 군의 기억·위키 구조와 회사 업무 스킬 현황을 3D force-directed 그래프로 시각화한 단일 HTML임. 노드를 클릭하면 스킬 원이름·용도·기능 및 효과가 상세 패널에 뜸.

## 공유 링크

**https://pujune.github.io/agent-system-3d-map/**

브라우저에서 위 링크를 열면 렌더된 3D 지도를 바로 볼 수 있음. 드래그로 회전, 휠로 줌, 노드 클릭 시 상세 정보와 카메라 이동이 동작함.

## 구성

- `index.html` — 3D 지도 본체이자 GitHub Pages 배포 진입점. 라이브러리와 데이터가 모두 인라인된 단일 파일임
